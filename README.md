# 🦕 QuizMania

Quiz sobre dinossauros com sistema de login em PHP.

## Tecnologias
- PHP
- MySQL
- HTML/CSS

## Como rodar localmente
1. Instalar XAMPP
2. Clonar o projeto na pasta `htdocs`
3. Criar banco de dados `quizmania` no phpMyAdmin
4. Criar tabela `usuario` com os campos: id, nome, genero, email, celular, senha
5. Ajustar `conexao.php` com suas credenciais
6. Acessar `localhost/quizmania`
